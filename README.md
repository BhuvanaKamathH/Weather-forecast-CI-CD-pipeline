# Weather-forecast-CI/CD-pipeline
Overview:
This project demonstrates an automated CI/CD pipeline built using Jenkins for a Java-based application.

Features:
Automated build and test pipeline using Jenkins
Dependency management with Maven
Integration with Slack for real-time notifications

Tech Stack:
Java
Jenkins
Maven
Git
Slack

Description:
The pipeline automates the process of building, testing, and monitoring the application, reducing manual effort and improving efficiency

Future Improvements:
Add deployment stage
Improve test coverage
